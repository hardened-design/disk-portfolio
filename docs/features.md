# 予定機能・ギミック

「High-Tech, Lo-Fi」と「情報の削ぎ落とし」を維持しつつ、実用性と遊び心を両立させる以下の機能を実装する。

## インタラクティブ・アスキーアート
*   **内容**: マウス操作による3Dオブジェクトの回転や、視点変更。
*   **根拠**: ユーザーが直接「触れる」体験を提供することで、Three.jsによるモダンな技術力と、アスキーアートのレトロな質感をダイレクトに伝える。

## ターミナル型パスコード入力（Locked Projects用）
*   **内容**: 公開制限のあるプロジェクトを表示する際、ターミナルUIが出現し、簡易的なパスコード入力を求める。
*   **根拠**: ナード/ハッカー文化へのオマージュ。単なる入力フォームではなく、世界観に合わせた「儀式」として提供することで、機密性の高い実績への関心を高める。

## デザイン・ガイドライン・トグル（Designer Mode）
*   **内容**: サイト全体のグリッド、ベースライン、余白の単位（px等）を可視化するオーバーレイ表示の切り替え機能。
*   **根拠**: グラフィックデザイナーとしての「論理的思考力」と「UI設計の緻密さ」を視覚的に証明する機能。審美眼の裏側にある設計意図を、同業者や採用担当者にアピールする。
