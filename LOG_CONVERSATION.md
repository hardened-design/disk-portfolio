## 1. サイトコンセプト: 「High-Tech, Lo-Fi（洗練された制約）」
*   **コアバリュー**: 「情報を削ぎ落として最大限の工夫をすることへの美しさ」

*   **ビジュアル**: 2chのアスキーアート、ゲームボーイ、Y2Kなどのナード/レトロ文化を、現代の技術（Astro/Three.js）で再構築する。

## 2. 設計思想と根拠（Rationale）

### CSS命名規則: **Astro Scoped CSS + Minimal**
*   **決定**: BEMなどの冗長な規則は採用せず、コンポーネント単位のスコープを活かした短い命名（`.wrapper`, `.canvas`等）を採用。
*   **理由**: 
    1.  Astroが自動でハッシュを付与するため、BEMによる衝突回避は冗長。
    2.  「情報の削ぎ落とし」というコンセプトをコードレベルでも体現する。
    3.  「昔のカオス（グローバルCSS）」を「現代の技術（Scoped CSS）」で制御する対比を重視。

### 技術スタック
*   **Framework**: Astro (Vanilla JS focus)
*   **3D Engine**: Three.js + AsciiEffect
*   **Styling**: Vanilla CSS (Scoped)

## 3. 予定機能・ギミック
*   **インタラクティブ・アスキーアート**: マウス操作による3Dオブジェクトの回転。
*   **ターミナル型パスコード入力**: 公開制限のある実績（Locked Projects）へのアクセス用。ナード/ハッカー文化へのオマージュ。
    *   **セキュア・モザイク**: CSSのぼかしではなく、サーバー/ビルド側で低解像度化・ディザリング（1bit風）した画像を生成。認証成功まで本画像を配信しない設計。コンセプトに合わせ「Dithered Pixel Art」や「ASCII Mosaic」として表現。
*   **デザイン・ガイドライン・トグル**: グリッドや余白を可視化。デザイナーとしての「論理的思考力」を視覚化する。
